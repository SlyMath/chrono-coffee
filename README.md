## Fonctionnalités
**Chronomètre :** Bouton Start et Stop.
**Barre de progression visuelle :** Animation fluide basée sur le temps écoulée.
**Mode Sombre :** Bouton permettant d'accéder au Dark Mode. 
Sauvegarde de la préférence utilisateur (LocalStorage).
